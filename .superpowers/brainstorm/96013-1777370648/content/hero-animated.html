⚡ Color Inheritance

The red gradient overlay, tag borders, and eyebrow line all read from theme_palette.accent — Burgito Red #D60523. A different project gets its own gradient automatically.

⚡ GSAP Utils in Use

gsap.utils.mapRange converts scroll 0→1 into image scale 1.08→1.0 and title translateY 0→-80px. clamp bounds the opacity so it never goes negative. normalize feeds the fill bar progress.

⚡ New DB Fields Shown

project.heroSummary was completely unused before — now it anchors the bottom strip. project.heroLabel drives the eyebrow. All overview[] facts appear in the right column.
